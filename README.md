# Tomofumi Torii — Portfolio

個人ポートフォリオサイト: https://www.tomofumitorii.com

## 構成

- `index.html`: ポートフォリオ本体（10 言語対応 i18n 組込）
- `assets/`: 画像・ロゴ等

## ローカル確認

```bash
open index.html
```

または:

```bash
python3 -m http.server 8080
# → http://localhost:8080
```
